Control Registers (32 bits, System Level)
===========================================================================
```
CR0 - System Control (PE, MP, EM, TS, ET, PG)
CR1 - Reserved
CR2 - Page Fault Linear Address
CR3 - Page Directory Base Register
CR4 - Extensions (PSE, PAE, PGE, etc.)
```
